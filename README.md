## Local Setup

Clone the repo:

```bash
git clone https://github.com/Phanhie/Mkdocs---Docs-Site.git
cd Mkdocs---Docs-Site
````

Install dependencies:

```bash
pip install mkdocs-material
```

Run the site locally:

```bash
mkdocs serve
```
